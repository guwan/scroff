- 关闭屏幕命令

<br />

<br />

<br />

```sh
adb shell "cd /sys/kernel/debug/dispdbg; echo disp0 > name; echo blank > command; echo 1 > param; echo 1 > start"
```

<br />

打开屏幕命令

<br />

```sh
adb shell "cd /sys/kernel/debug/dispdbg; echo disp0 > name; echo blank > command; echo 0 > param; echo 1 > start"
```
